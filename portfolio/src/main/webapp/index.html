About me

Projects

I have worked on many projects at hackathons. Here are a few of my favorites!

  • AsisText

    AsisText is a virtual assistant that works over SMS and MMS. This allows the assistant to be used without internet. The assistant can be used to look up images, wikipedia articles, set alarms and other things. This app won Booz Allen Hamilton Most Game Changing Hack at HackUMBC.

  • GotGas

    GotGas is a website that uses the Google Maps api to get users directions to a destination. The website gets users the most fuel efficient route instead of the fastest route.

  • Dinner Party

    Dinner party is a website that allows users to form dinner party groups. The website will then organize who will cook dinner each day while respecting each members schedule. The website will keep everyone up to date on who's cooking and what's for dinner.
